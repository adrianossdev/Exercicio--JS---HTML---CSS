Observações:
Olá,
desenvolvi este projeto utilizando CSS,HTML e a biblioteca VueJS.
Fiz meu 100%, porém, infelizmente, no tempo que eu calculei não consegui atender a todos os requisitos da prova, mesmo assim gostaria de apresentar o que consegui desenvolver.

Requisitos atendidos:
1- É uma aplicação de página única;
2- Listagem de produtos (localStorage);
3- Cadastro de produto (50%/localStorage);
4- Remoção de produto (localStorage);
5 - Visualização de produto (50%/localStorage);
6 - Dados persistidos sobrevivem ao reload.

Requisitos não atendidos:
1 - Layout foi desenvolvido para navegador Web (faltou adequar a usabilidade para mobile)
2 - Edição de produtos.
3 - Exibição da foto do produto e categorias(50%).
4 - Cadastro de produto - fotos e categorias(50%).
